Arduino Electrónica Attiny Oled

- Resumen -

El ATtinny85 es muy practico y funcional a pesar de sus restricciones debido a su tamaño, sin embargo con unos cuantos dispositivos es capaz de hacer funcionar motores, conectores Bluetooth y displays OLED

Este proyecto final es la agrupacion de todo lo visto en la materia de Elementos programables y su laboratrio, al usar un ATtinny85 para mover via Bluetooth un auto de 5cm x 5cm x 5cm

- Introducción -

El avance de la tecnología ha permitido que dispositivos electrónicos pequeños, como el ATtiny85, se utilicen para proyectos innovadores y creativos. Este proyecto consiste en convertir un automóvil Hot Wheels en un vehículo controlado de manera inalámbrica mediante Bluetooth, integrando un ATtiny85 como cerebro principal del sistema y un módulo Bluetooth para la comunicación.

El objetivo es construir un prototipo funcional que permita controlar el movimiento del vehículo utilizando una aplicación en un smartphone o cualquier dispositivo compatible con Bluetooth. Este proyecto no solo combina conceptos de programación, electrónica y comunicación inalámbrica, sino que también demuestra cómo es posible reutilizar objetos cotidianos para crear sistemas electrónicos modernos y versátiles.

MIT App Inventor es una plataforma en línea gratuita y accesible que permite a usuarios de todos los niveles de experiencia diseñar y programar aplicaciones móviles de manera visual e intuitiva. Desarrollada por el Massachusetts Institute of Technology (MIT), la herramienta utiliza un enfoque basado en bloques, similar a un rompecabezas, para construir la lógica de las aplicaciones, eliminando la necesidad de conocimientos avanzados en programación tradicional.

- Desarrollo -

Se diseñó un sistema compacto que incluyera: ATtiny85: Microcontrolador para procesar las señales recibidas por Bluetooth y controlar los motores del vehículo. HC-05 (o HC-06): Módulo Bluetooth para la comunicación entre el vehículo y el dispositivo móvil. Motores DC: Para el movimiento de las ruedas. Driver de motor L293D o similar: Para controlar la dirección y velocidad de los motores. Baterías recargables o una pila compacta: Para alimentar el circuito y los motores.

Conexión del ATtiny85: Se programó el microcontrolador con un Arduino Uno como ISP, configurándolo para recibir comandos desde el módulo Bluetooth y controlar el driver de motor. Módulo Bluetooth: Se conectaron los pines TX y RX del HC-05 a los pines PB0 y PB1 del ATtiny85 para la comunicación serie.

Desarrollo de la aplicación móvil con MIT App Inventor: Se creó una aplicación con botones básicos para las funciones de avanzar, retroceder, girar izquierda, girar derecha, y un control deslizante para ajustar la velocidad del vehículo. La funcionalidad de conexión Bluetooth se implementó utilizando el componente "BluetoothClient" de App Inventor, que permitió el emparejamiento con el módulo HC-05 del automóvil. Cada botón fue configurado para enviar comandos específicos, como adelante, atrás,izquierda y derecha, al microcontrolador.

Integración con el ATtiny85 y el circuito del vehículo: El ATtiny85 recibió los comandos a través del módulo HC-05 y procesó las señales para accionar los motores mediante el driver. Los valores de velocidad enviados por el control deslizante fueron convertidos en señales PWM para ajustar la potencia de los motores.

Simulación

Diseño

Diseño físico: Los componentes electrónicos (ATtiny85, HC-05, driver de motor y batería) se montaron dentro del chasis del automóvil Hot Wheels, utilizando cinta adhesiva y pegamento para asegurar su posición. Se optimizó el espacio para mantener la estética del vehículo mientras se garantizaba la funcionalidad de las ruedas y motores.

Aplicación móvil: Diseñada con MIT App Inventor, la aplicación envía comandos específicos (como "adelante", "atrás", "izquierda", "derecha") al HC-05 para controlar las acciones del vehículo. Un control deslizante permite enviar valores numéricos al ATtiny85 para ajustar la velocidad de los motores mediante PWM.

Diseño del circuito eléctrico: Alimentación: Las baterías proporcionan una fuente de 5V para el ATtiny85, el módulo Bluetooth y el driver de motor.

Conexiones del módulo Bluetooth HC-05: VCC → Línea de 5V de la batería. GND → Línea de tierra común del circuito. TX → PB0 (pin 5) del ATtiny85 para la recepción de datos. RX → PB1 (pin 6) del ATtiny85 para el envío de datos.
